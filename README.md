# AnimeViz
Тут короче пайтон от УРФУ прости господи
[![kartinka Image](5.png)](https://animeviz-ro2fu4xf4xyfmraruusssz.streamlit.app)

1. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

2. Run the app

   ```
   с пайтона streamlit run app.py
   если используете visual studio code то используйте python -m streamlit run app.py
   ```
